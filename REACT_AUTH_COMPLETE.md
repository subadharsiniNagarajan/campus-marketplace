## 🚀 To Run the App:

1. Install dependencies (this may take a few minutes):
   cd react-auth
   npm install

2. Make sure backend is running:
   cd ../backend
   npm start

3. Start React app:
   cd ../react-auth
   npm start

4. Open browser to http://localhost:3001
